# SpecSPDS

##  Autoloader

Автоматический загрузчик адаптеров nanoCAD  в зависимости от версии CAD 

[Единая "точка входа" (DLL файл) в плагин .NET, ARX или VBA, не зависящая от версии AutoCAD](https://bushman-andrey.blogspot.com/2014/06/dll-autocad.html)

С минимальной адаптацией заработало под nanoCAD

- .arx надо будет заменить на nrx
- .dvb нана не умеет


## Прикручено

для просто загрузчика это все излишества, но разобраться научится как это работает совсем нелишне

- nLog
- SimpleInjector


## Проблемы

- System.Management в загрузчике net462, в загружаемых сборках net6, в зависимости какой будет подтянут первым другой не работает.

Решение не использовать менеджер в загрузчике
